# Security policy

Companion is a multi-tenant control plane that handles authentication, organization data, deployment
credentials, and encrypted secrets. Please do not disclose suspected vulnerabilities in a public issue.

## Reporting a vulnerability

Use GitHub's private vulnerability reporting for this repository. Include the affected revision, a minimal
reproduction, the expected security boundary, and the observed impact. Do not include real credentials or
customer data.

The maintainers will acknowledge a complete report, assess its severity, and coordinate remediation and
disclosure with the reporter. There is no public bug-bounty program.

## Supported versions

Companion is pre-MVP. Security fixes are made on `main`; there are no supported release branches yet.
